opendefinition-org-all-licenses.json
------------------------------------
This file was procured from
https://licenses.opendefinition.org/licenses/groups/all.json and is used by
CKAN to provide a list of licenses that can be attached to datasets.  See
https://opendefinition.org/licenses/api/ for more information about the format
and how it's used.  The CKAN api docs discussing this is at
https://docs.ckan.org/en/latest/maintaining/configuration.html#licenses-group-url.
